Descrição do Algoritmo: O trabalho prático 01 consiste na criação de um programa que permita que o usuário
faça a utilização das Estruturas de Pilha e Fila Estática. Para isso o Usuário deve trabalhar
com a criação de Menu e Sub-menus, utilizando teclas de Atalho, e setas para percorre-los.
O menu Principal do Programa deve ter as opções: Aluno, Pilha, Fila e Sobre.
As estruturas Pilha e Fila devem ser vetores de 10 elementos do tipo inteiro cada.
Na opção Aluno deve ter o Menu: Cadastrar dados e Listar Dados.
Os dados de cadastramento são: Nome Completo, RGA, Endereço, E-mail, Nome pai,
Nome mãe, Celular, Data Nascimento, Cidade, Bairro, Curso, Semestre, Campus.
Nas opções de Estruturas (Pilha e Fila) devem conter o Menu: Inserir, Excluir, Listar
e Buscar.
No menu Listar deve-se ter o sub-menu: Ordem Crescente e Ordem Decrescente.
No menu Buscar deve-se ter o sub-menu: Busca Sequencial e Busca Binária. Retorne
a posição onde se encontra o número.
Para fazer a Busca Binária use um vetor auxiliar para receber os dados da Pilha/Fila,
ordene e somente depois você utilize a busca Binária. Para mostrar a posição do
elemento, verifique no vetor original a posição correta.
No menu Sobre deve conter um Resumo sobre o arquivo. Nome Programa, Nome
do Aluno, data e hora.
